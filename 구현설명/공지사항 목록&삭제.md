# 📌공지사항 목록/삭제 페이지   

![공지 목록삭제](https://user-images.githubusercontent.com/88878686/182356728-d29eceb7-ceef-4876-900c-11fa763a55e0.JPG)   

**페이지 당 10개**의 공지를 확인할 수 있도록 구현함.

확인을 클릭하면 해당 공지가 사라짐.
