# 📌유저관리   

![유저관리 목록](https://user-images.githubusercontent.com/105382798/182324946-0d031f80-64b0-4a64-83f6-a20a5bcb61be.png)

한 페이지에 **10명의 유저 확인 가능하며, 각 회원의 정보들도 확인 가능**
